# Stock Market Predictor using Monte Carlo Simulation



A Python-based stock market prediction system developed as a Class 12 Computer Science project. The application imports historical stock market data from CSV files into a MySQL database, performs statistical analysis, and uses a Monte Carlo Simulation model to estimate possible future stock price movements.



### Features



* Import historical stock data from CSV files into MySQL
* Automated CSV parsing using Python
* Organized MySQL database structure
* Statistical analysis of historical price data
* Monte Carlo Simulation for future price estimation
* Modular Python architecture
* Data visualization and graphical analysis *(Work in Progress)*
* HTML/Flask based user interface *(Planned)*



### Technologies Used



* Python 3
* MySQL
* MySQL Connector for Python
* CSV Module
* Math Module
* Random Module
* Flask *(Planned)*
* HTML \& CSS *(Planned)*





### How It Works



1\. Import historical stock data from CSV files.

2\. Store the data in a MySQL database.

3\. Retrieve historical closing prices.

4\. Calculate statistical parameters such as mean return and volatility.

5\. Run multiple Monte Carlo simulations.

6\. Display predicted price ranges and probability-based outcomes.



#### Note

Configure your MySQL credentials inside the database connection module before running the application.
